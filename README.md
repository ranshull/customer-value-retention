# Customer Value & Retention Analysis

## Overview
This project analyzes e-commerce customer behavior to identify revenue drivers, engagement patterns, and churn risk. The goal is to support business decisions related to membership strategy, discounting, and customer retention.

## Dataset
- 350 unique customers
- Synthetic dataset for illustrative purposes
- Includes demographics, spend, satisfaction, and recency metrics

## Tools Used
- Python (Pandas)
- SQL (DuckDB)
- Tableau

## Key KPIs
- Average Customer Spend
- Average Recency (Days Since Last Purchase)
- % At-Risk Customers
- Satisfaction Distribution
- Membership Value Lift
- Discount Impact

## Key Insights
- Gold members generate significantly higher value and engagement
- Discounts reduce average spend without improving satisfaction
- Unsatisfied customers show much higher churn risk
- Silver members represent the highest churn-risk segment

## Dashboard
An interactive Tableau dashboard was created to visualize KPIs and insights.

- `customer_value_retention_analysis.ipynb` – Analysis notebook
